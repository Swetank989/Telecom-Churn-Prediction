Project Title :Telecom churn prediction
Description: This project comapres Logistic Regression , Random Forest and XGBoost on the Kaggle customer churn dataset.
Dataset Source: Kaggle customer churn (https://www.kaggle.com/datasets/barun2104/telecom-churn)
Models Used: Logistic Regression , Random Forest , XGBoost
Metrics Comparison used : Confusion matrix , Accuracy score, roc_auc score along with roc_auc plot
